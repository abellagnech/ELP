Ceci est un programme en JavaScript qui implémente un shell simplifié en ligne de commande : il permet à l'utilisateur d'exécuter des commandes système de base:
-"lp" pour lister les processus en cours d'exécution
- "exec" pour exécuter un programme avec un chemin absolu
-"bing" pour tuer, mettre en pause ou reprendre un processus
-"help" pour afficher une aide sur les commandes disponibles
-"ctrl+P" pour sortir du shell.
Pour lancer le shell, il suffit de taper node shell_JS.js
N.B:
- CTRL + P qui permet de quitter le programme est une commande sur VSCode, il faut donc faire cette commande dans un terminal.
